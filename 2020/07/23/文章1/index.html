    ## 二、其中涉及到的环境变量
  • 在博客的建立中由于电脑本身环境的复杂，所以有时会因为环境变量的问题出错。
  • 环境变量是一般是指在操作系统中用来指定操作系统运行环境的一些参数，像：临时文件夹位置和系统文件夹位置等。所以在搭建博客的过程中若改变文件夹的位置就会出现找不见文件无法运行下去的现象。
    在安装hexo时就有
    ![3e331098de871489fa1903ee022a610272fa172b](3e331098de871489fa1903ee022a610272fa172b.png)
  • 出现时，有极大的可能是环境变量没有设置好。这时要返回去查看自己的环境变量是否按所需要求设置好了，像刚才那种情况可以通过 set 将 hello.exe 所在目录加到环境变量中。
